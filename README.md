# Spotify Clone 

A frontend Spotify-inspired web application built using **HTML, CSS, and JavaScript**.  
This project was created to practice **UI design**, **layout structuring**, and **basic JavaScript functionality**.

---

##  Features
- Spotify-like user interface
- Responsive layout for different screen sizes
- Music player UI (play / pause buttons)
- Playlist and song list design
- Clean and modern design

---

##  Technologies Used
- **HTML5** – structure of the web pages  
- **CSS3** – styling and responsive design  
- **JavaScript (Vanilla)** – basic interactivity
